# mock-apiendpoint

[streamlit-chat-ui](https://github.com/yoshiyuki-140/streamlit-chat-ui)の動作確認用のモックバックエンド

# 使い方

1. 仮想環境を起動

    ```bash
    uv venv
    source .venv/bin/activate
    ```

1. 依存関係を解消する
    ```bash
    uv pip install -r requirements.txt
    ```

1. 起動

    ```bash
    make run
    ```
